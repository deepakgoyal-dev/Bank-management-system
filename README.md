# Bank-management-system
Bank Management system built in python using Object-Oriented Programming(OOP)

## Features
-Create multiple bank accounts
-User login using account number,username and password
-Credit money
-Debit money
-Check account balance
-View transaction history
-Menu-driven interface

## OOP Concepts used:
-Classes and Objects
-Encapsulation
-Instance Attributes
-Methods
-Modules ('main.py','bank.py','accounts.py'
## Technologies used
-Python
